实名制平台通常用户质量更高。

真实用户越多，线下活动越丰富。

### 3、观察社区氛围

长期活跃的社区往往更加可靠。

### 4、保护个人隐私

不要轻易透露身份证、银行卡等重要信息。

正规的交友关系不会以借钱或转账作为开始。
